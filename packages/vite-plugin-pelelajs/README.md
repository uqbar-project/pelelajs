# vite-plugin-pelelajs

The official Vite plugin for **PelelaJS**. It provides template transformation and HMR support for a seamless development experience.

## Installation

```bash
pnpm add -D vite-plugin-pelelajs
```

## Usage

Add it to your `vite.config.ts`:

```typescript
import { defineConfig } from 'vite'
import { pelelajsPlugin } from 'vite-plugin-pelelajs'

export default defineConfig({
  plugins: [pelelajsPlugin()]
})
```
